Wp-map-markers

A WordPress plugin to display Google Maps on your website. Uses the Store Locator library and API v3

Download this project as a .zip file Download this project as a tar.gz file

Welcome to WP Map Markers.

WP Map Markers is a WordPress plugin based on the Google Maps API v3. It allows you to add Store Locator type maps to your posts and pages. You could use it to display your store or office branches, your photos if you have a photoblog, anything really.

Features

Support for multiple maps (not on the same page / post though) Different marker icons ( currently 6 color variations, more to come) Street view Autocomplete search Directions Feature filtering List of nearby locations
